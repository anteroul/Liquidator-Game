# Liquidator-Game
An unfinished game project made with Raylib and Go.

### Controls:
- Arrow keys = movement
- Spacebar = fire weapon
- Numeric keys 1-5 = select weapon
- R = reload weapon
- P = pause game
- F1 = toggle frame limiter on/off
- Esc = Exit game
